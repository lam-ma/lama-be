Lama Backend
==
Hack week (day?) project

API:
-
[OpenApi spec](/openapi.yaml)

Prerequisites:
-
Project requires JDK 11+ installed 

How to run
-

To build self-contained jar:
```./gradlew shadowJar```

To run self-contained jar:
```java -jar build/libs/lama.jar```

To run from gradle: 
```./gradlew run```
